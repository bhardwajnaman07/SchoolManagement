1. Install Python(3.7.6) (Dont Forget to Tick Add to Path while installing Python)

2. Open Terminal and Execute Following Commands :

	python -m pip install -r requirements.txt

3. Move to project folder in Terminal. Then run following Commands :

	py manage.py makemigrations
	py manage.py migrate
	py manage.py runserver

4. Now enter following URL in Your Browser Installed On Your Pc

	http://127.0.0.1:8000/
